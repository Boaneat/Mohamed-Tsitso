Object Oriented Programming

Python is known as an object object oriented programming

This means it is easy ti represent entities as objects by using classes and encapsulation

An example of a this might be represented by Room class to represent a room 

                
                    # create a class
                    class Room:
                        length = 0.0
                        breadth = 0.0
                        
                        # method to calculate area
                        def calculate_area(self):
                            print("Area of Room =", self.length * self.breadth)

                    # create object of Room class
                    study_room = Room()

                    # assign values to all the attributes 
                    study_room.length = 42.5
                    study_room.breadth = 30.8

                    # access method inside class
                    study_room.calculate_area()

We use this class by calling the class
